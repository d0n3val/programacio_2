Changes

The primary reason for creating this offline version was to be able to include help with a Devcpp package of SDL2. It would also be usefull for places without internet connection, or with expensive or limited connection speeds.

This offline simplified version of the wiki has been manually typed in and edited. As such there may be errors, and the documentation will age as the real wiki gets updated.

If I ever get unlimited broadband, I'll attempt to automate the conversion process, but as that is not likely, this documentation will be updated as and when I have time.

Simplification

The main result of simplification is removal of syntax highlighting in code examples (it just would take one person too long), and removal of the contents list on each page. Also any wiki editing features obviously were removed.

To cater for files being supplied by file system instead of web server, the file names all end in html.

To cater for operating systems which are not case sensitive (think MS Windows® all versions, and DOS), the structures as identified in the API By Name have been put into a subfolder structures amd the links adjusted accordingly.
